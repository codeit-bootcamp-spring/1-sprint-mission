## 웹 API 요구사항

### 사용자 관리

[ ] 사용자를 등록할 수 있다.

[ ] 사용자 정보를 수정할 수 있다.

[ ] 사용자를 삭제할 수 있다.

[ ] 모든 사용자를 조회할 수 있다.

[ ] 사용자의 온라인 상태를 업데이트할 수 있다.


### 권한 관리

[ ] 사용자는 로그인할 수 있다.


### 채널 관리

[ ] 공개 채널을 생성할 수 있다.

[ ] 비공개 채널을 생성할 수 있다.

[ ] 공개 채널의 정보를 수정할 수 있다.

[ ] 채널을 삭제할 수 있다.

[ ] 특정 사용자가 볼 수 있는 모든 채널 목록을 조회할 수 있다.


### 메시지 관리

[ ] 메시지를 보낼 수 있다.

[ ] 메시지를 수정할 수 있다.

[ ] 메시지를 삭제할 수 있다.

[ ] 특정 채널의 메시지 목록을 조회할 수 있다.


### 메시지 수신 정보 관리

[ ] 특정 사용자의 메시지 수신 정보를 조회할 수 있다.
